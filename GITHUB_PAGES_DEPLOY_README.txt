Quick deploy (no terminal required):
1) Create a new public GitHub repository (empty).
2) Open the repo in browser, click Add file > Upload files.
3) Upload ALL files/folders from this package, then Commit changes.
4) Go to Settings > Pages.
5) Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
6) Save. GitHub will publish your site URL in 1-3 minutes.

Optional: use a custom domain in Settings > Pages.

Notes:
- This app is fully static and secure over HTTPS by default on GitHub Pages.
- If a change does not appear, hard refresh (Ctrl+F5).
